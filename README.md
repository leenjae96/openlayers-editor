# 📝 Openlayer vertes editor (버텍스 수정 어플리케이션)
**OpenLayers 기반의 인터랙티브 공간 데이터 편집 도구입니다.** <br/>
지도상에서 점, 선, 면 등 지리 정보의 정점(Vertex)을 실시간으로 수정하고 관리할 수 있습니다. <br/>
프론트엔드 개발 과정에서 Gemini와 Codex 등 생성형 AI 도구를 적극 활용하여 핵심 로직을 빠르게 프로토타이핑하고, 복잡한 공간 데이터의 시각화와 사용자 상호작용을 구현하는 데 집중했습니다.
추후 확장 계획: Spring Boot와 MySQL을 연동하여 데이터 관리 예정.
